# Sound Effects

This directory should contain the following sound files for the interactive portfolio:

- `click.mp3` - Button click sound effect
- `hover.mp3` - Hover sound effect
- `success.mp3` - Success notification sound
- `whoosh.mp3` - Transition/scroll sound effect

## Sound Requirements

- Format: MP3
- Duration: 0.1-0.5 seconds
- Volume: Low/subtle (will be set to 30% in code)
- Quality: Web-optimized, small file size

## Recommended Sources

You can find suitable sound effects from:

- Freesound.org
- Zapsplat.com
- Adobe Audition presets
- Or create custom sounds

## Note

The website includes a sound toggle button, so users can disable audio if preferred. All sounds are designed to enhance the user experience without being intrusive.
