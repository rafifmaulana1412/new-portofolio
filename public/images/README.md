## How to Replace

1. Take screenshots of the actual websites
2. Resize images to 600x400px (or similar aspect ratio)
3. Optimize images for web (compress to reduce file size)
4. Replace the placeholder files with the same filename
5. The portfolio will automatically use the new images

## Image Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress images
- Maintain aspect ratio of 3:2 (width:height)
- Keep file sizes under 200KB for better loading performance
- Use PNG for images with text/UI elements
- Use JPG for photographic content

## Fallback

If actual screenshots are not available, the portfolio will display placeholder images with project names.
